--User requirements for analyser tool--

1. The User is able to present a data file that can be read by the program.
2. The User should be able to easily read the report generated by the program.
3. The User should be able to easily view graphical models on student ranking and modules.
4. The User should be shown a clear indication of modules that appear too easy/too hard.
5. The User is able to filter the data to find the rank of strongest/ weakest student within each module/all modules.
